      McDonald's Console Application 🍔
=>Project Description

This is a Java console-based McDonald's ordering system created using Core Java.
The application allows users to select food items from a menu, add them to a cart, and generate a final bill.
This project was built to practice Core Java concepts like Scanner, Switch Case, Loops, and Functions.

=>Features

Display McDonald's menu
Select food items
Add items to cart
Quantity selection
Calculate total bill
Simple console interface

=> Technologies Used

Java
Core Java Concepts
Scanner Class
Switch Case
Loops
Functions

=>How to Run the Program

1)Clone the repository
2)Open the project in any Java IDE (Eclipse / IntelliJ / VS Code)
3)Compile the program
4)Run the main class
Example command:
      javac McDonalds.java
      java McDonalds
